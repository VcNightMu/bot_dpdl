### 1. resolver/ — 精确匹配

#### operation.py
```python
def resolve_operation(user_input: str, menu_index: dict) -> str | None:
    """精确匹配关卡名，返回operation或None"""
    # 1. 精确匹配operation（大小写不敏感）
    key = user_input.strip().upper()
    if key in menu_index["by_operation"]:
        return menu_index["by_operation"][key]
    
    # 2. 精确匹配cn_name
    if user_input.strip() in menu_index["by_cn_name"]:
        return menu_index["by_cn_name"][user_input.strip()]
    
    return None
```

### 3. 视频链接处理（在formatter中实现）

```python
import re

def format_url(url: str) -> str:
    """格式化视频链接：B站截取BV号，非B站显示'非B站视频'"""
    if not url:
        return "非B站视频"
    match = re.search(r'(BV[\w]+)', url)
    if match:
        return match.group(1)
    return "非B站视频"
```
